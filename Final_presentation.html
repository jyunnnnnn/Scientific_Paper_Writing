Challenge 01

Incompatibility

Element-wise ops vs. Tensor Core

Tensor Cores are built for reduction. Mamba's element-wise ops don't need reduction, leading to massive inefficiency.

Strategy: Using numbers to prove inefficiency.

Challenge 02

Area Overhead

Wasted silicon area

Specialized units for nonlinear functions (like Exp) occupy too much space but are not always used.

Strategy: Criticizing resource waste.

Challenge 03

Memory Bottleneck

Existing methods fail

High memory access but limited data reuse makes traditional 'Tiling' optimization impossible.

Strategy: Using the strong word 'Useless'.
